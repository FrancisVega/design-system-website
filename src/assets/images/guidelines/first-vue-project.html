Buttons

Buttons use text and/or an icon to express what action will occur when the user clicks or touches it. Buttons are used when an action takes place, either in the background or foreground of an experience.

Variations

Primary Buttons

Primary buttons should be used for the principal call to action on the page. What is the most important action your user should take? Primary buttons should be three words or less.
