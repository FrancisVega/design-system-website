## Typography 

Tab text should be set in sentence case, and should not exceed three words.

| PROPERTY | FONT-SIZE       | FONT-WEIGHT  |
|----------|-----------------|--------------|
| Tab text | 14px / 0.875rem | Bold / 700   |
