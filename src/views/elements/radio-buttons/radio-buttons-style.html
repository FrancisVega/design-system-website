### Basics Radio buttons have two states, selected and unselected. Selected buttons are filled by a small, solid dot encircled by a 2px border. The dot and border should be the same color. Unselected buttons have a 2px border with no fill.

## Typography Number input labels should be set in bold and sentence case, with only the first word in a phrase and any proper nouns capitalized. Number input text should be set in roman and only contain numbers, no letters. All number input type should follow the Bluemix Typography font stack for san-serif.
