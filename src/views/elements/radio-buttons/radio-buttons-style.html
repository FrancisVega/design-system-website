### Basics Radio buttons have two states, selected and unselected. Selected buttons are filled by a small, solid dot encircled by a 2px border. The dot and border should be the same color. Unselected buttons have a 2px border with no fill.

### Padding Radio buttons cannot have any element or text within 16 pixels of its borders. There should be 16px / 1rem between the radio button and the radio button label text.

## Typography Number input labels should be set in bold and sentence case, with only the first word in a phrase and any proper nouns capitalized. Number input text should be set in roman and only contain numbers, no letters. All number input type should follow the typography font stack for sans-serif.
