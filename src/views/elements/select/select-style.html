{{#markdown}} ## Structure
### Basics Select boxes are single color filled at 10% opacity. The width of a select dropdown box should appropriately fit the design and layout of the content. **Select Box Background Color: rgba(255,255,255) at 10% opacity** **Select box height: 40px / 2.5em**
### Padding **Space between label and select box: 10px / .625em** **Internal side padding: 16px / 1em**

Select Basics | units in px / em

### States - Default Non-Selected State: Browser Defined - Hover: 1px blue outline (rgb 90, 170, 250) - Open: Style determined by browser - New Selection:__ - Form Validation: 2px bottom border (rgb 255, 80, 80) - Help: Tool tip should be left aligned with the text field box and underlined to indicate a link. Disabled: 50% opacity

Select States

### Groupings Select dropdowns can be grouped horizontally and vertically for tablet and desktop views. Select dropdowns cannot be grouped horizontally for mobile views. **Padding between vertical groupings: 32px / 2em** **Padding between horizontal groupings: 16px / 2em**

Select Groupings Orientations | units in px / em

## Typography
Select labels should be set in title case, with the first letter of every word capitalized. Labels should be 3 words or less. Select dropdown text should be set in sentence case, with only the first letter of the first word capitalized. Select dropdown text should follow the Bluemix Typography font stack for sans-serif. **Select Label font-size: 14px / .875rem** **Select Label font-weight: bold / 700** **Select Dropdown Text font-size: 14px / .875rem** **Select Dropdown Text font-weight: roman / 400**
