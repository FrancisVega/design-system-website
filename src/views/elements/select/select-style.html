## Typography
Select labels should be set in sentence case, with only the first word in a phrase and any proper nouns capitalized. Labels should be 3 words or less. Select dropdown text should be set in sentence case, with only the first word in a phrase and any proper nouns capitalized. Select dropdown text should follow the Typography font stack for sans-serif.
