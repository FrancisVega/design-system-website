### Padding Tags cannot have any element or text within 3px / .188 em of its borders. Internally, there must be 10px / .625em of space between the tag text and the sides of the tag.

### Typography Tag text should be set in title case with the first letter of each word capitalized. Tags should only have one word. However, if more than one is necessary then connect the words using a hyphen with no spaces. Text should follow the typography font stack for sans-serif.
