### Basics The height of an uploaded file is set at 30px / 1.875 em. The width of an uploaded file however, varies based on the content and layout of a design. The uploaded file is accompanied by a remove icon which can be found in the Bluemix Icon Repo. Refer to the button style guidelines for styling of the 'add files' button. **Form width: Varies based on content and layout**
Please refer to the text field, dropdown, checkbox, radio button, toggle and button pages for additional styling and usage details. All icons can be found in the Bluemix Icon Repo or the Iconography Library page.

### Typography The File Uploader Title should be set in title case with the first letter of each word capitalized. File Uploader instruction text should be set in sentence case, with only the first letter of the first word in the sentence capitalized. File name text should appear exaclty as it was originally inputed by the user, except when shortenign it with the ellipse. File uploader text should follow the Typography font stack for san-serif.
