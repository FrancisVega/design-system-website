## Typography
Checkbox legend text should be set in **bold / 700** and sentence case, with only the first word in a phrase and any proper nouns capitalized. Checkbox label text should be set in **regular / 400** and sentence case, with only the first word in a phrase and any proper nouns capitalized. Checkbox labels should not exceed more than 3 words. Checkbox text should follow the typography font stack for sans-serif.
