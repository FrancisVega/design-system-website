{{#markdown}} ## Structure
### Basics Checkboxes have two states: selected (checked) and not selected (unchecked). Selected or checked boxes are single-colored and filled with a contrasting colored check mark. Unselected or unchecked boxes have a 2px border with no fill. **Checkbox height and width: 18px / 2.7em** **Checkbox outline: 2px border** **Space between checkbox and checkbox label text: 8px / .05em** **Space between checkboxes: 16px / 1em**

Checkbox | units in px / em

### Padding A checkbox cannot have any element or text within 16 pixels / 1 em of its borders. **External Padding: 16px / 1em**
## Typography
Checkbox legend text, or titles, should be set in bold / 700 and title case, with the first letter of each word capitalized. Checkbox label text should be set in regular / 400 and sentence case, with only the first letter of the first word capitilized. **Legend font size: 14px / .875em** **Legend font weight: bold / 700** **Label font size: 14px / .875em** **Legend font weight: normal / 400**
