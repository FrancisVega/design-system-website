### Basics Checkboxes have two states: selected (checked) and not selected (unchecked). Selected or checked boxes are single colored filled with a contrasting color for the check mark. Unselected or unchecked boxes have a 2px border with no fill. **Checkbox height and width: 18px / 2.75em** **Checkbox outline: 2px border**

### Padding Checkboxes cannot have any element or text within 16 pixels / 1 em of its borders. **Padding between checkbox and checkbox label text: 8px / .05em** **Padding between checkboxes: 16px / 1em**
## Typography
Checkbox legend text should be set in bold / 700 and title case, with the first letter of each word capitalized. Checkbox label text should be set in regular / 400 and sentence case, with only the first letter of the first word capitilized. Checkbox should not exceed more than 3 words. Checkbox text should follow the Bluemix Typography font stack for sans-serif. **Legend font size: 14px / .875rem** **Legend font weight: bold / 700** **Label font size: 14px / .875rem** **Legend font weight: normal / 400**
