{{#markdown}} ## Structure
### Basics Toggles have two states: action on and action off. When an action state is on, the toggle button should be solid and on the right end of the toggle bar. When an action state is off, the button should have a 2px border with no fill and be on the left end of the toggle bar. The font for the unactive state should be lighter than the active state. **Toggle Button height & width: 24px / 2.5em** **Button Border: 2 px / .125 em** **Font size: 14px / .875 em** **Font weight: normal / 400**

Toggle | units in px / em

### Padding Toggles can not have any element or text within 16 pixels / 1em of its borders. **External Padding: 16px**

Toggle Padding | units in px / em

### Typography Toggle text should be set in title case, with the first letter of each word capitalized. Toggle labels should be 3 words or less. Toggle text should follow the Bluemix Typography font stack for sans-serif.
