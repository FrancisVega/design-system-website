### Basics Toggles have two states: action on and action off. When an action state is on, the toggle button should be solid and on the right end of the toggle bar. When an action state is off, the button should have a 2px border with no fill and be on the left end of the toggle bar. The font for the unactive state should be lighter than the active state.

### Typography Toggle headings should be set in title case, with the first letter of each word capitalized. Toggle labels should be set in sentence case, with only the first word in a phrase and any proper nouns capitalized, and no more than 3 words. Toggle text should follow the Typography font stack for sans-serif.
