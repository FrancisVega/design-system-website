### Basics The height of a module always stays the same. The width does not have a set size, however, it should fit within the Bluemix Grid. Meaning a module should start or end before or after a gutter, not in the middle of a column or gutter. The modules should be modular so that they can stack.

## Typography
Module headers should be set in bold and sentence case, with only the first word in a phrase and any proper nouns capitalized. Module headers should not exceed more than 3 words. Module internal labels should also be set in sentence case and should not exceed more than 3 words. Module content text should be set in sentence case, with only the first word in each sentence and any proper nouns capitalized. Data text should be set in title case with the first letter of each word capitalized. All module type should follow the Bluemix Typography font stack for sans-serif.
