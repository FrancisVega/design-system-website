## Typography
Level 1 of any list type should be set in bold and sentence case, with only the first word in a phrase and any proper nouns capitalized. Level 2 of any list type should also be set in sentence case, with only the first word in a phrase and any proper nouns capitalized in roman. All list text should follow the Bluemix Typography font stack for san-serif.
