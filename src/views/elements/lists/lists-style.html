## Typography
Level 1 of any list type should be set in title case, with the first letter of every word capitalized and bolded. Level 2 of any list type should also be set in title case, with the first letter of every word capitalized in roman. All list text should follow the Bluemix Typography font stack for san-serif. **List font-size: 14px / .875rem** **Level 1 List font-weight: bold / 700** **Level 2 List font-weight: roman / 400** **Line height: 24px**
