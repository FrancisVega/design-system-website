{{#markdown}} ## Structure
### Basics There are two types of lists: ordered and unordered. Level 1 ordered list points are marked numerical with Level 2 points marked with a dash. Level 1 unordered list points are bulleted with Level 2 points marked with a dash.

Ordered List | units in px / em

Unordered List | units in px / em

### Padding Lists cannot have any element or text within 16 px / 1em of its borders. **External Padding: 16px / 1em**

Ordered List Padding | units in px / em

Unordered List Padding | units in px / em

## Typography
Level 1 text in a list should be set in bold/700 and title case, with the first letter of every word capitalized. Level 2 text in a list should be set in roman/400 and title case as well. List text should follow the Bluemix Typography font stack for sans-serif. **List font-size: 14px / .875em** **Level 1 List font-weight: bold / 700** **Level 2 List font-weight: roman / 400** **Line height: 24px**
