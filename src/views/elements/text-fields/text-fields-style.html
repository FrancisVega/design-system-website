### Basics Text fields are single color filled at 10% opacity. The width of a text field should appropriately fit the design and layout of the content. **Text Field Background Color:** **Dark UI:** #ffffff at 10% opacity rgba(255, 255, 255, .10) **Light UI:** #3d70b2 at 10% opacity rgba(61, 112, 178, .10)

### States - Default Non-Selected State: Browser Defined - Hover: 1px outline #5aaafa (rgb 90, 170, 250) - Form Validation: Dark UI: 2px bottom border #ff5050 (rgb 255, 80, 80) Light UI: 2px bottom border #e71d32 (rgb 231, 29, 50) - Help: Tool tip should be left aligned with the text field box and underlined to indicate a link - Disabled: 50% opacity

### Groupings Text fields can be grouped horizontally and vertically for tablet and desktop views. Text fields cannot be grouped horizontally for mobile views. **Padding between vertical groupings: 32px / 2em** **Padding between horizontal groupings: 16px / 2em**

## Typography
Text field labels should be set in title case, with the first letter of every word capitalized and should not exceed more than 3 words. Text field text should be set in sentence case, with only the first letter of the first word capitalized. Text fields text should follow the Bluemix Typography font stack for sans-serif.
