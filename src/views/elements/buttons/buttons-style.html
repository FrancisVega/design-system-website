### Basics Primary buttons are a single color fill with a contrasting color for the text of the label. Secondary buttons have a solid 2px border with no fill. The text and border color on secondary buttons styles are the same color. **Button height: 40px / 2.5em** **Small Button height: 32px / 2em** **Padding left and right of text: 16px / 1em** **Font size: 14px / .875 rem** **Font weight: bold / 700** ### Padding A button cannot have any element or text within 16 pixels / 1 em of its borders. **Internal side padding: 16px / 1em** **Button external padding: 16px / 1em** **Small button external padding: 10px / .625em**

## States
### Primary Style **Focus:** Browser Defined **Hover:** Filled and one shade lighter on the IBM Design Language color scale than normal color. Text stays the same color. **Disabled:** 50% opacity. ### Secondary Style **Focus:** Browser Defined **Hover:** Filled and the same color as the stroke in the normal state. Text should change to a contrast color. **Disabled:** 50% opacity.
## Typography
Button text should be set in title case, with the first letter of each word capitalized. Button labels should not exceed more than 3 words. Button text should follow the Bluemix Typography font stack for sans-serif. **Font-size: 14px / .875rem** **Font-weight: bold / 700**
