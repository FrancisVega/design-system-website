### Basics The width of tool tip is always the same at 240px / 15em however the height varies based on the amount of content. The tool tip icon can be found in the Bluemix Icons Repo. **Tool tip height: varies based on amount of content** **Tool tip width: 240px / 15em** **Tool tip icon height and width: 16px / 1em** **Tool tip icon color: #5AAAFA** **Tool Tip box drop shadow: #000000 50% Opacity** **Tool Tip box pointer: 8px / .5em**

### Padding Internally there must be 24px / 1.5em of space between the tool tip text and sides of the tool tip. Padding around the tool tip icon is 10px / .625em **Padding between header and icon: 10 px / .625em** **Padding between icon and tool tip: 10 px / .625em** **Internal Padding: 24px / 1.5em**

### Placement Tool tips should always be placed below the tool tip logo unless it is within the height of the tool tip box plus 16px / 1em of the bottom of the page. In that instance, the tool tip should appear to the right of the tool tip icon. On mobile, tool tips can only appear below the tool tip icon. **Tool tip placement below icon: bottom of tool tip must be at least 16px / 1em from the bottom of page**

### Typography Tool tip labels should be set in title case with the first letter of each significant word capitalized. Tool tip text should be in sentence case with only the first word of every sentence capitalized. Text should follow the Bluemix Typography font stack for sans-serif. **Tool Tip Label font-size: 14 px / .875rem** **Tool Tip Label font-weight: bold / 700** **Tool Tip Text font-size: 14 px / .875rem** **Tool Tip Text font-weight: roman / 400**
