### Basics Search boxes are a single color fill with a contrasting color for the text of the label. The width of the search field should appropriately fit the design and layout of content. Larger search fields should include a filter button with a label and filter icon. Smaller search fields should have a filter button with a filter icon and no label. **Search Box Color: Dark UI: #ffffff at 10% opacity rgba(255, 255, 255, 0.10) Light UI: #3d70b2 at 10% opacity rgba(61, 112, 178, 0.10)** **Search Box Height: 50px / 3.125em** **Small Search Box Height: 40px / 2.5em** **Filter Icon (height & width): 16 px / 1em** **Search Icon (height & width): 20 px /1.25em** **Small Search Icon (height & width): 20px / 1.25em**
### Padding **Space between box and search icon: 12px / .75em** **Space between search icon and search text: 12px / .75em**

## Typography
Search text should be set in sentence case, with only the first letter of the first word capitalized. Search text should follow the Bluemix Typography font stack for sans-serif. **Large Search Field Normal and Hover font-size: 14px / .875rem** **Large Search Field Normal and Hover font weight: roman italic / 400** **Large Search Field Input font-size: 14px / .875rem** **Large Search Field Input font-weight: bold / 700** **Filter Button font-size: 14px / .875rem** **Filter Button font-weight: bold / 700**
