{{#markdown}} ## Structure
### Basics Links can be grouped horizontally or vertically and must be underlined. **Font size: 14px / .875rem** **Font weight: bold / 700** **Line height: 24px / 1.5em** **Space between each link (horizontally): 18px / 2.75em** **Line height: 24px / 1.5em**

Horizontal and vertical Link styling

### Link States Links have three different states, normal, hover, and disabled which are indicated through color change. - Browser Defined - Hover: One shade lighter on the IBM Design Langauge color scale than normal color. - Disabled: 50% opacity
## Typography
Links should be set in title case, with the first letter of each word capitalized. Links should be 3 words or less. Links should follow the Bluemix Typography font stack for sans-serif.
