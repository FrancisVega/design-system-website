### Link States Links have three different states, normal, hover, and disabled which are indicated by change in color. - Browser Defined - Hover: One shade lighter on the IBM Design Langauge color scale than normal color. - Disabled: 50% opacity
## Typography
Links should be set in title case, with the first letter of each word capitalized. Links should not exceed more than 3 words. Links should follow the Bluemix Typography font stack for sans-serif. **Font size: 14px / .875rem** **Font weight: bold / 700**
