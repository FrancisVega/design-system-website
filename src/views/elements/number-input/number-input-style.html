### Basics The height of a number input is always the same. The width however, varies based on content, layout and design. The arrows icon can be found in the Bluemix Iconography Library or the Bluemix Icon Repo. **Number input width: varies based on content, layout, and design** **Number input height: 40px / 2.5em** **Dark Number input color: #FFFFFF at 10% opacity** **Light Number input color: #3D70B2 at 10% opacity**

### Padding There should be 16px / 1em of internal padding on the left and right side of the number input. **Padding between left side and text: 16px / 1em** **Padding between right side and arrows: 16px / 1em**

## Typography Number input titles should be set in bold and title case, with the first letter of each word capitalized. Number input text should be set in roman and only comtain numbers, no letters. All number input type should follow the Bluemix Typography font stack for san-serif. **Number Input title font-size: 14px / .875rem** **Number Input title font-weight: bold / 400** **Number Input text font-size: 14px / .875rem** **Number Input text font-weight: roman / 400**
