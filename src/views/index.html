IBM Bluemix
Design System

Thoughtful, detailed guidelines that enable teams to confidently design delightful experiences.

Design Kit Developer Kit

The Bluemix Design System provides transparency into the Bluemix Component Library, core visual styles, interaction patterns. Using Bluemix best practices, we want to enable others to design and build experiences that contribute to our united platform. This set of living guidelines is constantly evolving and growing as we learn from our users, and most importantly, from you.

Base Elements

Consider base elements as the deepest level of the Design System. These are very specific UI elements and patterns that make up a larger experience, such as buttons, links, and checkboxes.

Components

Certain base elements are combined and resued based on the experience or the context. Components are composed of 1 or more base elements, such as tables, cards, and activity logs.

Templates

Templates provide a base structure for specific pages built within Bluemix. Use these as a starting point and foundation for your design to be built around.

IBM Design Language
