General

Text

The text within an overflow menu should be direct so users can quickly decide on an action.

Actions that could cause a significant change to the user's data (Delete app, Delete Service, etc), should be separated by a horizontal rule and live below the primary set of actions.

Overflow menu in Cards

Due to the space constraints, it is often not possible to contain all relevent actions to manipulate an object within the card. An overflow menu may be used to present additional menu options that directly relate to the card.
