Tables are a configurable and customizable component. Designers can pick and choose certain elements and interactions. Table widths can vary based on the layout, amount of data and content. ### Basics The basic table style is the required base from which tables can start developing. Basic tables are composed of a header row followed by rows of data below. At a minimum, tables require 3 or more columns. Table row heights can be set to narrow or wide or be toggled between the two. Column widths can vary by content and only requires a minimum spacing between columns.
