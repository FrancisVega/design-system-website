General

Dropdown lists are used when the selection of items are of secondary importance. If space is a concern, use dropdowns to minimize distraction.

Text

Dropdown content should be limited to a single line of text that describes the action it will perform when selected. Do not use multiple lines of text, images, or icons within a dropdown. Sort the items in logical order, such as grouping related options together.

Length

Dropdown options should be as succinct as possible. Long dropdown menus that require scrolling make it difficult for users to see all their options within one glance.

Interaction

Dropdowns should appear on-click, rather than on hover. Hover-activated dropdowns increase cognitive load on users and are not recommended for use due to usability purposes.

Dropdown menus should appear above all other elements within the UI. They can be dismissed by clicking outside of the menu item or on the parent element. Generally, selecting an item from the dropdown menu will also dismiss the menu.

Structure

Basics

Dropdown menus have two major states, open and closed. The open and closed menu should be the same width and appropriately fit the design, layout, and content. The height of closed menu should always be 40px / 2.5em. The height of an open menu will vary based on the amount of menu options it has. Please note the various color differences for closed and open menus.

Dropdown Menu height: 40px / 2.5em Dropdown Menu width: varies based on content and layout

Closed Dropdown Menu:

Background color: white at 10% opacity (rgba: 255, 255, 255, 0.15)

Open Dropdown Menu:

Background color: white at 10% opacity (rgba: 255, 255, 255, 0.15)
Dropdown menu bar color: #3A4B54

Padding

Internally there must be 16px / 1em of space between the sides of the dropdown menu and the elements within the menu.

Internal side padding: 16px / 1em

Dropdown Padding | units in px / em

States

In addition to the open and closed states, dropdown menus have hover, new selection, and disabled states.

Hover border: 1px
Hover color: #5AAAFA
Disabled: 50% opacity

Typography

All dropdown menu text should be set in title case with the first letter of each word capitalized. Dropdown menu options should not exceed more than 3 words. Text should follow the Bluemix Typography font stack for sans-serif.

Dropdown menu label font-size: 14 px/ .875rem
Dropdown menu label font-weight: bold / 700
Dropdown menu option text font-size: 14px /.875rem
Dropdown menu option text font-weight: roman / 400
