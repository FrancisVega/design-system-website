{{#markdown}} ## General The waiting experience is a crucial design opportunity. Although it may not be obvious what is occurring on the back-end, we can communicate clearly to reassure the the user that progress is happening. It is best practice to use a loading spinner whenever the wait time is anticipated to be longer than 3 seconds.
