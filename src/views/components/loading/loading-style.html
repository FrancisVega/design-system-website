{{#markdown}} ## Structure
### Basics There are two types of loading spinners, large and small. The large loading spinner appears with an overlay on top of a whole page of content. **Large Loading Spinner height and width: 84px / 5.25em** **Large Loading Spinner color: Teal 40 #1CB3A0** **Overlay background color: #0F212E at 75% opacity** **Small Loading Spinner height and width: 16px / 1em** **Small Loading Spinner color: Navy-Gray 7 #8C9BA5**

Large Loading Spinner | units in px / em

Small Loading Spinner | units in px / em

### Placement The large loading spinner should appear centered over a page or content that it is loading. Please note that the naviagation is not included in the page loading overlay. The small loading spinner should be placed to the left of a label. **Large Loading Spinner: centered in loading area** **Small Loading Spinner: 8px / .5em to the left of label**
