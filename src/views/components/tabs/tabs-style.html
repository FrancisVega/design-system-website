Text should follow the Bluemix Typography font stack for sans-serif. **Primary level tab options font-size: 14px / .875rem** **Primary level tab options font-weight: bold / 700** **Secondary level tab options font-size: 12 / .75rem** **Secondary level tab options font-weight: bold / 700**
