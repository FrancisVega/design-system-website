{{#markdown}} ## Structure
### Basics There are two different types of modals, passive and transactional. Passive modals contain information while transactional modals contian information and have actionable buttons in the bottom left hand corner. Refer to the button style guidelines for styling. On mobile the button/action bar is sticky to the bottom of the screen. Both passive and transactional modals can have a label above the title, however it is optional. **Modal top border: 5px** **Modal top border color: #5596E6** **Close button height and width: 10px / .625em** **Space between buttons: 16px / 1em** **Transactional button bar height: 92px / 5.75em**
For desktop and tablet: **Modal width: varies based on layout, design, and content** **Modal height: varies based on layout, design, and content**

### Padding Modals should have plently of breathing room around each element. See below for specific space measurements between elements. Modals on mobile should follow page margins of grid of 3%. **Top of modal to label: 44px / 2.75em** **Top of modal to title: 44px / 2.75em** **Padding between label and title: 8px / .5em** **Padding between title and body copy: 24px / 1.5em** **Padding between top of modal and close button: 16px / 1em** **Padding between right edge of modal and close button: 16px / 1em** **Padding between left/right edge of modal and modal text: 48px / 3em**
For Passive Modals only: **Space between body copy and bottom of modal: 48px / 3em**

## Typography
Modal Labels are optional and should be set in all caps when used. Modal Titles should be set in title case with the first letter of each word capitalized. Modal Titles should not exceed more than 3 words. Modal body copy text should be sentence case with only the first word of the sentence capitalized. Important directions or information in the body copy can be bold, othewise it should be in roman. Modal labels, titles, and text should all follow the Bluemix Typography font stack for sans-serif. **Modal Label font-size: 12px / .75rem** **Modal label font-weight: bold / 700 / all caps** **Modal Title font-size: 26px / 1.625rem** **Modal Title font-weight: roman / 400** **Modal body copy font-size: 14px / .875rem** **Modal body copy font-weight: bold / 700** **Modal body copy font-weight: roman / 400**
