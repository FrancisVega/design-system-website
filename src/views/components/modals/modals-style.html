### Basics There are two different types of modals, passive and transactional. Passive modals only contain information while transactional modals have information and actionable buttons on the bottom left-hand corner. Refer to the Button style guidelines for styling. On mobile, the button/action bar is sticky to the bottom of the screen. Both passive and transactional modals may have an optional label above its title.

## Typography
Modal Labels are optional and should be set in all caps when used. Modal Titles should be set in title case with the first letter of each word capitalized. Modal Titles should not exceed more than 3 words. Modal body copy text should be sentence case with only the first word of the sentence capitalized. Important directions or information in the body copy can be bold, othewise it should be in roman. Modal labels, titles, and text should all follow the Typography font stack for sans-serif.
