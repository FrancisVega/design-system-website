## Typography
Card names and card information should be set in title case with the first letter of each word capitalized. A card url should be set in all lower case letter. Status text should be set in title case with the first letter of each word capitalized. All card type should follow the Bluemix Typography font stack for sans-serif. ### App Card **Card name font-size: 18px / 1.125rem** **Card name font-weight: light / 300** **App url font-size: 12px / .75em** **App url font-weight:roman / 400** **App url font color: #5AAAFA** **Status text font-size: 12px / .75em** **Status text font-weight: roman / 400** ### Container Card **Card name font-size: 18px / 1.125rem** **Card name font-weight: light / 300** **Container information font-size: 12px / .75em** **Container information font-weight:roman / 400** **Container information font color: #5AAAFA** **Status text font-size: 12px / .75em** **Status text font-weight: roman / 400** ### Quota Card **All text alignment: centered** **Number font-size: 46px / 2.875em** **Number font-weight: light / 300** **Status information font-size: 14px / .875em** **Status information font-weight: roman / 400** ### Service Card **Card name font-size: 18px / 1.125em** **Card name font-weight: light / 300** **Service information font-size: 12px / .75em** **Service information font-weight:roman / 400** **Service information color: #8C9BA5** ### Virtual Servers Card **Card name font-size: 18px / 1.125em** **Card name font-weight: light / 300** **Virtual Server information font-size: 12px / .75em** **Virtual Server information font-weight:roman / 400** **Virtual Server information font color: #5AAAFA** **Status text font-size: 12px / .75em** **Status text font-weight: roman / 400**
