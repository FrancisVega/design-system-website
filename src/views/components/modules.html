Structure

Header

Headers provide consistency and context to the module. Keep the text short and precise. Buttons or links may be used within a header, and should always be right aligned.

Body

Footer

A footer can be used for content or actions that need to be persistent throughout the module.

Max Height

It is recommeded to design within the set height of a module. All content in a module needs to fit without internal scrolling.

Max Width

Modules have a max width in order to structurally fit within the context of the UI.

Empty States

In the case where a module has no data to display, an empty state must be shown that gives the user guidance on next steps. It is recommended to include an actionary item such as a button or a link that gives the user further direction.

Structure

Basics

The height of a module is always the same. The width, does not have a set size, however it should fit within the Bluemix Grid. Meaning a module should start or end before or after a gutter, not in the middle of a column or gutter.

Module Height: 300px / 18.75em
Module width: flexible but should fit to grid

View at 100%
Modules fit over the Bluemix Grid
Modules | units in px / em

Padding

Padding between modules, both above/below and on the sides, should be 20px / 1.25em, the same width as the gutter on the Bluemix grid. The modules should be modular so that they can stack.

Internal Padding: 24px / 1.5em
Padding between Modules: 20px / 1.25em

Modules Padding | units in px / em

Typography

Notification titles should be set in bold and title case with the first letter of each word capitalized. Notification titles should not exceed more than 3 words. Notification text should be set in sentence case with only the first word capitalized. All notification type should follow the Bluemix Typography font stack for sans-serif.

Header font-size: 14px / .875rem
Header font-weight: bold / 700
Internal Labels font-size: 12px / .75rem
Internal Labels font-size: bold / 700
Internal Labels character spacing: 1px
Module text font-size: 14px / .875rem
Module text font-weight: roman / 400
Data Text font-size: 32px / 2rem
Data Text font-weight: light / 300
