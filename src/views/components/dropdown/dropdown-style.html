### Basics Dropdown menus have two major states, open and closed. The open and closed menu should be the same width and appropriately fit the design, layout, and content. The height of a closed menu should always be 40px / 2.5rem. The height of an open menu will vary based on the amount of menu options it has. Please note the various color differences for closed and open menus.

## Typography
All dropdown menu text should be set in title case with the first letter of each word capitalized. Dropdown menu options should not exceed more than 3 words. Text should follow the Bluemix Typography font stack for sans-serif.
