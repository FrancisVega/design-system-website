### Basics Dropdown menus have two major states, open and closed. The open and closed menu should be the same width and appropriately fit the design, layout, and content. The height of a closed menu should always be 40px / 2.5rem. The height of an open menu will vary based on the amount of menu options it has. Please note the various color differences for closed and open menus. **Dropdown Menu height: 40px / 2.5rem** **Dropdown Menu width: varies based on content and layout** Closed Dropdown Menu: **Background color:** **Dark UI**: #ffffff at 10% opacity (rgba: 255, 255, 255, 0.10) **Light UI**: #3d70b2 at 10% opacity (rgba: 61, 112, 178, 0.10) Open Dropdown Menu: **Background color:** **Dark UI**: #ffffff at 10% opacity (rgba: 255, 255, 255, 0.10) **Light UI**: #3d70b2 at 10% opacity (rgba: 61, 112, 178, 0.10) **Dropdown drawer background color:** **Dark UI**: #3a4b54 **Light UI**: #ffffff Small Dropdown ##### Height: 24px / 1.5rem ##### Width: 56px / 3.5rem

### States In addition to the open and closed states, dropdown menus have hover, new selection, and disabled states. Dropdown: **Hover border: 1px** **Hover color: #5aaafa** **Disabled: 50% opacity** **New Selection color fill: #5aaafa** Small Dropdown Whichever number is displayed at the top of the dropdown box, the corresponding number in the list below should have a check mark next to it, indicating which number is the one currently selected. The icon for the check mark can be found in the Iconography Library or the Bluemix Icons Repo. **Hover (Dark): 1px solid outline, #5AAAFA** **Hover (Light): 1px solid outline, #3D70B2** **Open hover (Dark): #5AAAFA** **Open hover (Light): #5596E6** **Disabled: 50% opacity**

## Typography
All dropdown menu text should be set in title case with the first letter of each word capitalized. Dropdown menu options should not exceed more than 3 words. Text should follow the Bluemix Typography font stack for sans-serif. **Dropdown menu label font-size: 14 px/ .875rem** **Dropdown menu label font-weight: bold / 700** **Dropdown menu option text font-size: 14px /.875rem** **Dropdown menu option text font-weight: roman / 400**
