{{#markdown}} ## Structure
### Basics Dropdown menus have two major states, open and closed. The open and closed menu should be the same width and appropriately fit the design, layout, and content. The height of closed menu should always be 40px / 2.5em. The height of an open menu will vary based on the amount of menu options it has. Please note the various color differences for closed and open menus. **Dropdown Menu height: 40px / 2.5em** **Dropdown Menu width: varies based on content and layout** Closed Dropdown Menu: **Background color: white at 10% opacity (rgba: 255, 255, 255, 0.15)** Open Dropdown Menu: **Background color: white at 10% opacity (rgba: 255, 255, 255, 0.15)** **Dropdown menu bar color: #3A4B54**

Dropdown Basics | units in px / em

### Padding Internally there must be 16px / 1em of space between the sides of the dropdown menu and the elements within the menu. **Internal side padding: 16px / 1em**

Dropdown Padding | units in px / em

### States In addition to the open and closed states, dropdown menus have hover, new selection, and disabled states. **Hover border: 1px** **Hover color: #5AAAFA** **Disabled: 50% opacity** **New Selection color fill: #5AAAFA**

Dropdown States

## Typography
All dropdown menu text should be set in title case with the first letter of each word capitalized. Dropdown menu options should not exceed more than 3 words. Text should follow the Bluemix Typography font stack for sans-serif. **Dropdown menu label font-size: 14 px/ .875rem** **Dropdown menu label font-weight: bold / 700** **Dropdown menu option text font-size: 14px /.875rem** **Dropdown menu option text font-weight: roman / 400**
