### States There are four states for notifications: warning, informational, error, and success. The notification state is indicated by icon and border color. This rule applies to all three types of notifications (inline, toast, and modal). ## Typography
Notification titles should be set in bold and title case with the first letter of each word capitalized. Notification titles should not exceed more than three words. Notification text should be set in sentence case with only the first word capitalized. All notification type should follow the Bluemix Typography font stack for sans-serif. **Notification title font-size: 14px / .875rem** **Notification title font-weight: bold / 700** **Notification text font-size: 14px / .875rem** **Notification text font-weight: roman / 400**
