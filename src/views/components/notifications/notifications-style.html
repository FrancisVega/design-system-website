### Basics There are three types of notifications: inline, generic, and modal. Please refer to the usage tab when determining notification type. The border color of a notification and its accompanying icon indicate the notification state. The left border of a notification should be 5px, while the remaining sides are 1px. Notifications on desktop and tablet should not exceed two lines, including the notification title. Notifications on mobile should not exceed three lines, including the notification title.

### States There are four states for notifications: warning, informational, error, and success. The notification state is indicated by icon and border color. This rule applies to all three types of notifications (inline, toast, and modal). All icons may be found in the Icon Repo or the Iconography page.

## Typography
Notification titles should be set in bold and title case with the first letter of each word capitalized. Notification titles should not exceed more than three words. Notification text should be set in sentence case with only the first word capitalized. All notification type should follow the Typography font stack for sans-serif.
