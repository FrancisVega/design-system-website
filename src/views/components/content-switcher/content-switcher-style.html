{{#markdown}} ## Structure
### Basics Content switchers must have at least two options for the user to choose from. Each container that makes up the content switcher is equal in size. The width of a container is determined by the length of the longest container option text plus the 32 px / 2em on both sides of the text. **Content Switcher height: 40px / 2.5em** **Content Switcher container width: varies based on amount of content** **Content Switcher fill color: #5AAAFA** **Border: 1px** **Border color: #5AAAFA**

Content Switcher | units in px / em

### States Content switchers have two states: selected and not selected. Selected states are a solid color. There must always be only one selected state, no more, no less. Unselected options have 1px border with no fill.
### Padding Internally there must be 32px / 2em of space between the content container option text and sides of the content container. **Padding between side of content box and text: 32px / 2em**

Content Switcher Padding | units in px / em

## Typography
Content switcher text should be set in title case with the first letter of each word capitalized. Content Switcher option text should not exceed more than 3 words. Text should follow the Bluemix Typography font stack for sans-serif. **Content switcher font-size: 14px / .875em** **Content switcher font-weight: roman / 400**
