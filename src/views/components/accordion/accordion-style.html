### Basics Accordion rows can be open or closed. A user can have multiple rows open at once. There is no limit to the height of an open row, however the padding specs below should be followed. The width of an accordion varies based on the content, layout, and page design. The chevron icon can be found in the Bluemix Icon Repo or on the Iconography Library page. The chevron has two different colors, one when the row is closed, and a second for when the row is open.

### Padding There must not be any text within 16px / 1em of the accordion's sides. Additionally there must be padding between labels, body copy, and seperation lines to provide breathing room between elements.

### Typography All accordion labels should be set in title case, with the first letter of each word capitalized. Labels hsould not exceed three words. Body copy text should be set appropriately based on content. All text should follow the Bluemix Typography font stack for san-serif.
