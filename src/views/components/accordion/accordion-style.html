### Basics A user may have multiple rows open at once. There is no limit to the height of an open row, however, the padding specs below should be followed. The width of an accordion varies based on the content, layout, and page design. The chevron icon may be found in the Icon repo or on the Iconography library page. The chevron has two different colors, one when the row is closed and a second for when the row is open.

### Padding Avoid having any text within 16px / 1rem of the accordion's sides. Additionally, include padding between labels, body copy, and separation lines to provide breathing room between elements.

### Typography All accordion labels are set in title case, with the first letter of each word capitalized. Labels should not exceed three words. Set body text appropriately based on content. All text should follow the typography font stack for sans-serif.
