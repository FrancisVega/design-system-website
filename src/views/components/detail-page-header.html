General

The Detail Page Header, which appears at the top of the page, is used to allow users to navigate throughout the views of Bluemix. The header uses Tabs as a primary navigation pattern.

Example:

View at 100%

Structure

Basics

The width of detail headers extends across the entire browser, therefore the width of the detail page header is determined by the width of the browser.

When a user first comes to a Detail Page the header should be full height of 148px / 9.25em, when the user starts to scroll down the page it should shrink to 64px / 4em. The header is sticky to the top of the page.

On mobile the height of the header bar stays consistant at 135px / 8.438em. Please refer to the dropdown section for additional guidelines on styling dropdown menus.

Detail Page Header for desktop and tablet:

Detail Page Title height: 148px / 9.25em
Detail Page Title width: varies based on browser size

Detail Page Header for scrolling:

Detail Page Title height for scrolling: 64px / 4em
Detail Page Title width: varies based on browser size

Detail Page Header for mobile:

Detail Page Title height: 135px / 8.438em
Tab menu height: 40px / 2.5em

Padding

Each of the three detail page header variations have very specific padding requirements. Please see below for the details. The back arrow icon can be found in the Bluemix Icons Repo.

Detail Page Header for desktop and tablet:

Padding between top of header and Breadcrumb Title: 32px / 2em
Padding between Breadcrumb Title and Detail Page Title: 16px / 1em
Padding between Detail Page Title and Tab Options: 24px / 1.5em

Tab Options and bottom of header: 16px / 1em
Padding between left side of header and internal elements: 68 px / 4.25em
Padding between Back Arrow Icon and Breadcrumb Title: 8px / .5em
Padding between tabs: 48px/ 3em

Padding between top and bottom of header and internal elements: 16px / 1em

Padding between Back Arrow Icon and divider: 16px / 1em
Padding between divider and Detail Page Title: 16px / 1em

States

The detail page header for mobile uses a tab menu that has two states, open and closed. Additionally when the menu is open there is a hover state for when users mouse over a tab menu option. Please refer to the dropdown section for additional guidelines on styling dropdown menus.

Typography

Detail page titles, breadcrumb titles, and tab labels should all be set in title case with the first letter of each word capitalized. Text should follow the Bluemix Typography font stack for sans-serif.

Detail Page Header for desktop and tablet:

Detail Page Title font-size: 32px / 2em
Detail Page Title font-weight: light / 300
Breadcrumb Title font-size: 14px / .8765em
Breadcrumb Title font-weight: bold / 700
Tab Label font-size: 14px / .8765em
Tab Label font-weight: bold / 700
