### Basics The overflow menu should always have a width of 180px / 11.25em. The height of an overflow menu is determined by the amount of content in the menu. The overflow menu icon can be found in the Bluemix Icons or on the Iconography Library page. **Overflow menu height: varies based on content** **Overflow menu width: 180px / 11.25em** **Line height: 32px / 2em** **Dividing Line border: 1px** **Dividing Line color:** **Dark UI:** #5a6872 **Light UI:** #dde2e6

### States Open overflow menus have a hover state that is indicated by a color change. All hover states should be #5aaafa (Light UI: #3d70b2) unless the option the user is hovering over cannot be undone, in which case the color is #ff5050 (Light UI: #e71d32). Menu option text color changes during a hover state. **Regular hover color:** **Dark UI:** #5aaafa **Light UI:** #3d70b2 **Warning hover color:** **Dark UI:** #ff5050 **Light UI:** #e71d32 **Text hover color:** **Dark UI:** #152934 **Dark UI:** #ffffff

### Padding Internally, there must be 16px / 1em of space between the menu text and the ssides of the overflow menu. **Internal Padding: 16px / 1em** **Padding between menu icon and overflow menu: 8px / .5em**

## Typography
Overflow menu text should be set in title case with the first letter of each significant word capitalized. Text should follow the Bluemix Typography font stack for sans-serif. **Overflow menu font-size: 14px / .875rem** **Overflow menu font-weight: roman / 400**
