{{#markdown}} ## Structure
### Basics The overflow menu should always have a width of 180px / 11.25. The height of an overflow menu is determined by the amount of content in the menu. The overflow menu icon can be found on the Bluemix Icons page. **Overflow menu height: varies based on content** **Overflow menu width: 180px / 11.25em** **Line height: 32px / 2em** **Dividing Line border: 1px** **Dividing Line color: #5A6872**

Overflow Menu | units in px / em

### States Open overflow menus have a hover state, indicated by a color change. All hover states should be #5AAAFA unless the option the user is hovering over cannot be undone, in which case the color is #FF5050. Menu option text color changes during a hover state. **Regular hover color: #5AAAFA** **Warning hover color: #FF5050** **Text hover color: #152934**

Overflow Menu States | units in px / em

### Padding Internally there must be 16px / 1em of space between the menu text and sides of the overflow menu. **Internal Padding: 16px / 1em** **Padding between menu icon and overflow menu: 8px / .5em**

Overflow Menu Padding | units in px / em

## Typography
Overflow menu text should be set in title case with the first letter of each significant word capitalized. Text should follow the Bluemix Typography font stack for sans-serif. **Overflow menu font-size: 14px / .875rem** **Overflow menu font-weight: roman / 400**
