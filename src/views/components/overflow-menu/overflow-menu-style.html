### States Open overflow menus have a hover state that is indicated by a color change. All hover states should be #5aaafa (Light UI: #3d70b2) unless the option the user is hovering over cannot be undone, in which case the color is #ff5050 (Light UI: #e71d32). Menu option text color changes during a hover state.

## Typography
Overflow menu text should be set in title case with the first letter of each significant word capitalized. Text should follow the Typography font stack for sans-serif.
