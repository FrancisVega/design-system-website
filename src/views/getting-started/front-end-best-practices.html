{{#markdown}} ## Maintainability IBM Design says, "focus on the user." Front end development bridges design and engineering, bringing content to reality and making products valuable for users. - IBM Design Language But, this also includes the engineers and front-end developers you work with — they are also users because we all share and rely on each other's code to build products for our end-users. Writing maintainable code ensures that we continuously fulfill our responsibilities to continue making essential, delightful products for our end-users. - Write readable and well-named code. - Write code that's well-documented. - Write testable code. The Bluemix Design System's technical documentation, including contribution and code guidelines, can be found here.
See the IBM Design Front End Guidelines for more guidance.
