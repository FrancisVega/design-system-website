Title

If necessary, a title may accompany a set of checkboxes to provide further clarity into what the user is making a selection on.

Labels

A checkbox should always be accompanied by a clear and concise label. Be explicit about the action that will follow if the checkbox is selected. Labels appear to the right of the checkbox.

Click Target

Users should be able to select the checkbox by clicking on the box directly or its label.

Tables

See Tables section for guidance on how to use checkboxes within a table.

Default Selection

A set of checkboxes should default to having none of the options selected.

Structure

Basics

Checkboxes have two states: selected (checked) and not selected (unchecked). Selected or checked boxes are single colored filled with a contrasting color for the check mark. Unselected or unchecked boxes have a 2px border with no fill.

Checkbox height and width: 18px / 2.7em
Checkbox outline: 2px border
Space between checkbox and checkbox label text: 8px / .05em
Space between checkboxes: 16px / 1em

Checkbox | units in px / em

Padding

Checkboxes cannot have any element or text within 16 pixels / 1 em of its borders.

External Padding: 16px

Typography

Checkbox legend text should be set in bold / 700 and title case with the first letter of each word is capitalized.

Checkbox label text should be set in regular / 400 and sentence case with only the first letter of the first word capitilized.

Legend font size: 14px / .875em
Legend font weight: bold / 700
Label font size: 14px / .875em
Legend font weight: normal / 400
