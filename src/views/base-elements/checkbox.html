Title

If necessary, a title may accompany a set of checkboxes to provide further clarity into what the user is making a selection on.

Labels

A checkbox should always be accompanied by a clear and concise label. Be explicit about the action that will follow if the checkbox is selected. Labels appear to the right of the checkbox.

Click Target

Users should be able to select the checkbox by clicking on the box directly or its label.

Tables

See Tables section for guidance on how to use checkboxes within a table.

Default Selection

A set of checkboxes should default to having none of the options selected.
