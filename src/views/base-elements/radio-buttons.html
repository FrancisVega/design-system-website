Title

If necessary, a title may accompany a set of radio buttons to provide further clarity into what the user is making a selection on.

Lables

A radio button should always be accompanied by a clear and concise label. Be explicit about the action that will follow if the radio button is selected. Labels appear to the right of radio buttons.

Default Selection

A set of radio buttons should default to having exactly one option selected. Never display them without a default selection.

Structure

Basics

Radio buttons have two states, selected and unselected. Selected buttons are filled by a small, solid dot encircled by a 2px border. The dot and border should be the same color. Unselected buttons have a 2px border with no fill.

There should be 16px / 1em between the radio button and the radio button label text.

Radio Button height and width: 16px / 1 em
Radio Button border: 2 px / .125 em
Smaller fill dot height and width: 8px / 0.5 em

Radio button | units in px / em

Padding

Radio buttons cannot have any element or text within 16 pixels of its borders.

External Padding: 16px / 1em

Radio button padding | units in px / em

Typography

Radio button legend text should be set in bold / 700 and title case with the first letter of each word capitalized.

Radio button label text should be set in regular / 400 and sentence case with only the first letter of the first word capitilized.

Radio button labels should not exceed more than 3 words. Radio button text should follow the Bluemix Typography font stack for sans- serif.

Radio Legend font size: 14px / .875 em
Radio Legend font weight: bold / 700
Radio Label font size: 14px / .875 em
Radio Label font weight: regular / 400
