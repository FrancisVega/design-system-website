Guidelines

Title

If necessary, a title may accompany a set of radio buttons to provide further clarity into what the user is making a selection on.


Labels

A radio button should always be accompanied by a clear and concise label. Be explicit about the action that will follow if the radio button is selected. Labels appear to the right of radio buttons.


Default Selection

A set of radio buttons should default to having exactly one option selected. Never display them without a default selection.


"Other"

If using the label "other" in a set of radio buttons, it's recommended to provide a text field for the user to input text to clarify their selection.
