Order

The order of the list should be based on the frequency of use. If applicable, the list should be in increasing order relative to the content. In cases of forms, alternative orders such as alphabetical may be more fitting. A horizontal rule can be used to group similar items together.

Accessibility

The default select element is completely usable either with a mouse or a keyboard, following this process:

  1. Mouse: move your cursor over the select element
    Keyboard: use the tab key to focus the select element
  2. Mouse: click on the select element
    Keyboard: press enter
  3. Mouse: move your cursor over the desired option
    Keyboard: use the top and bottom arrow keys to pick an option
  4. Mouse: click on the desired option
    Keyboard: press enter

While you can make a CSS dropdown easily usable by a mouse, making it usable with keyboard navigation is complex.
