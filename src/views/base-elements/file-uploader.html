General

File uploaders are commonly found in forms, but they can also live as stand alone elements.

File uploaders should always be accompanied by a Set of Buttons.

Stand alone uploaders

When using a file uploader as a stand alone element, a common pattern for sets of buttons is "Cancel/Upload".

Interaction

  1. The user may select 1 or more files to upload at a time
  2. The action of clicking "Choose File" will trigger a browser-specific upload window
  3. Once the user chooses files to upload, the text field should respond to this behavior. If one file is selected, the text field should display the name of the file. If more than one file is selected, the text field should read "# files selected"
