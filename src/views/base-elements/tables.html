Structure

Header Row

Headers guide users to the data that is presented below. Use meaningful text to label the table data and provide clarity to the data below. If necessary, use footnotes to clarify more detailed or technical information that would otherwise clutter the table. Headers should be fixed on scroll if the table is longer than the viewport size.

Number of Columns

Alternating colors on rows increases readability, and helps the guide the users through the data in the table.

Minimal

Usability of your table will be greatly improved by making data sortable. Sorting can be helpful when users are looking to compare adjacent information. By making header labels clickable, it allows the user to easily filter the data to their liking.

Zebra Stripes

Expanable rows allow for minimal information to be shown at a high level, and a more detailed description/action set hidden within the overflow of the row.

Expandable Rows

Expanable rows allow for minimal information to be shown at a high level, and a more detailed description/action set hidden within the overflow of the row. .

Continuous Scroll

Continuous scroll is used when presenting large amounts of data sets. Instead of directing a user to a different page that contains additional content, a simple, "View More" link that loads a more data is appropriate.
