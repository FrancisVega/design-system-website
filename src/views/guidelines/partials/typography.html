Typography is used to create clear hierarchy throughout the layout. It is the core structure of well designed interfaces. When type scaling is used appropriately it creates intentional flows, user focus, and progressive emphasis.

Typeface

IBM uses a custom version of Helvetica Neue for both its Marketing and Product brands. It is a strong but neutral typeface that lends personality to the page without overpowering the content. The font files for Bluemix can be found at https://github.ibm.com/Design/fonts/tree/master/fonts

Font weights

Font weight is another typographic style that can add emphasis and differentiate content in the hierarchy of the design. Font weight in conjunction with type size must be carefully balanced. At the same size, bold weight will always have more emphasis than a lighter weight font. However, a larger, lighter weight font is ranked higher than a smaller, bold font.

Headers

Size of headers are determined by layout or template structure. Bluemix has several levels of architecture and areas that require varying sizes of headers. Nevertheless, all headers must be sized from the Major Second type scale with a thoughtfully considered font weight.

Body Copy

Font Size

Bluemix uses two sizes for body copy. The first is UI specific. Due to valuable screen real estate we chose a smaller 0.875rem / 14px body copy size for the standard UI console. However, for areas that have prolonged reading, like Documentation, we use a larger body copy size of 1rem/16px to enhance readability.

Standard Console: 0.875rem / 14px
Prolonged reading: 1rem / 16px

Line-height

Line-height, traditionally known as leading, is one of several factors that directly contribute to readability and pacing of copy. Line-heights are based off the size of the font itself. Ideal line-heights for standard copy have a ratio of 1.5. For example, a type at 1em/16px would have a line-height of 24px (16 x 1.5). The exception to this rule is headings, which need less spacing and therefore have a line-height ratio of 1.25.

Standard line-height: 1.5
Headings line-height: 1.25

Line-length

Line-length, traditionally known as measure, is the number of characters in a single line. It also directly contributes to readability and pacing of copy. Lines that are too long degrade eye tracking from line to line, making it difficult to gauge which line to read next. In contrast, lines that are too short make it difficult for a reader to maintain a steady reading rhythm. Short lines also often create disproportionate ragged edges that negatively affect the design.

Line-length: 52 - 78 characters

Custom Styling

Letter-spacing

Custom letter-spacing is used only on a few special styles. The styles that it can be applied to are headers, bold weights and always to All Caps.

Letter-spacing: 0.5px

Font-smoothing

Font-smoothing is only used on bold weights (700). It is never used on thin (200) and light (300) weights.
