Dark UI Color Usage

The base background layer for Dark UI is Navy-Gray 3. Global Bluemix elements live on the darkest background (for example, the catalog and add menu), while the more detailed and specific content becomes progressively lighter. The primary accent color for Dark UI is Blue 30.

As you drill down into the product to the details and specifics the background gradually lightens
