Adding Icons

If you'd like to add an icon to our library, please submit a Pull Request in our GitHub Repo.

If you're not comfortable with making a Pull Request, please submit an issue in the repo with the icon attached.

  • Fork the GitHub repo.
  • Clone it to your local system.
  • Add icons in the proper folder (you may add your own) within the svg/ folder.
  • In your terminal run npm install.
  • Run npm run build to rebuild the sprite.svg.
  • Submit a Pull Request and rejoice!

Exporting SVGs

Best Practices:

  • Create an individual artboard for each icon at 32 x 32 pixels. Export the entire icon artboard. Do not trim trim transparent pixels.
  • Note: Icons can be resized in the code but the default SVG size should be 32 x 32 pixels.
  • Make sure each artboard and the artwork within is pixel perfect. There should not be decimals in the x and y coordinates or in the width and height fields.
  • Delete any excess styling. The icon should be exported at the color #000000. All styling (size, color, etc.) will be done in code.
  • Ungroup the icon layers completely. It should be on the top-most layer in your artboard.
  • Make sure to properly name layers and artboards (these names will also be exported into the code).

How to Export SVGs from Sketch

  1. Draw a 32 x 32 pixel stand-alone artboard.
  2. Place icon in the pixel perfect center of the artboard.
  3. Make sure icon is #000000 and has no additional styling.
  4. Select the entire artboard (not just the icon).
  5. Click "Make Exportable" at the bottom of the right toolbar in Sketch.
  6. In the "Export" widget select "SVG" in the format dropdown.
  7. Click "Export 'Artboard-Name'".
  8. Name icon appropriately.

How to Export SVGs from Adobe Illustrator

  1. Draw a 32 x 32 pixel stand-alone artboard.
  2. Place icon in the pixel perfect center of the artboard.
  3. Make sure icon is #000000 and has no additional styling.
  4. Select "File" from the top navigation.
  5. Select "Save A Copy..." which will open a dialog.
  6. On the "Format" dropdown select "SVG (svg)".
  7. Below "Format" select "Use Artboard" then select either all or a range of artboards depending on your need.
  8. Click "Save".
  9. The "SVG Options" dialog will then open.
